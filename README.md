# 🐍 贪吃蛇游戏

一个现代化的网页版贪吃蛇游戏，支持电脑端和移动端操作，具有友好的用户界面、智能的游戏机制和沉浸式音效体验。

## 🎮 游戏特色

### 🎯 核心功能
- **经典贪吃蛇玩法**：控制蛇移动，吃食物增长身体和得分
- **智能碰撞检测**：蛇头可以到达边界，只有穿过墙壁时才游戏结束
- **防误操作设计**：禁止反向移动，避免意外后退导致的游戏失败
- **网格显示**：清晰的网格线帮助玩家精确控制移动方向

### 🎚️ 游戏控制
- **9档速度调节**：从900ms到100ms，每档递减100ms，默认500ms
- **智能速度锁定**：游戏进行中锁定速度设置，防止意外调整
- **暂停/继续功能**：随时暂停游戏，支持中途休息
- **重新开始机制**：游戏结束后需手动点击开始，可在开始前调整设置
- **等待首次输入**：点击开始游戏后，等待用户首次方向操作才开始移动

### 🔊 音效系统
- **得分音效**：吃到食物时播放愉快的和弦音效 (C-E-G)
- **游戏结束音效**：失败时播放下降音调的提示音
- **静音控制**：右上角静音按钮，随时开启/关闭音效
- **Web Audio API**：使用现代音频技术，无需外部音频文件

### 📱 多平台支持
- **电脑端**：使用方向键 ↑↓←→ 控制蛇的移动
- **移动端**：触屏滑动控制，支持上下左右滑动手势
- **响应式设计**：自动适配不同屏幕尺寸

## 🚀 快速开始

1. 打开 `index.html` 文件
2. 调整游戏速度（1档最慢900ms，5档默认500ms，9档最快100ms）
3. 点击"开始游戏"按钮（此时速度被锁定）
4. 按方向键或滑动手势进行首次操作，蛇开始移动
5. 吃到红色食物增长身体和得分
6. 点击右上角🔊按钮可开启/关闭音效

## 🎯 游戏规则

### 基本规则
- 🍎 **吃食物**：碰到红色食物可以增长身体和得分，同时播放得分音效
- 🚫 **避免自撞**：不能撞到自己的身体
- 🧱 **边界规则**：蛇头可以到达边界格子，但不能穿过墙壁
- ⚡ **移动限制**：不能反向移动，防止意外后退

### 游戏设置
- 🎚️ **速度调节**：游戏开始前可调整，游戏中被锁定
- ⏸️ **暂停功能**：游戏进行中可随时暂停/继续
- 🔄 **重新开始**：点击重新开始后回到准备状态，可调整速度，需再次点击开始游戏
- ⏳ **等待首次输入**：点击开始游戏后等待用户首次方向操作才开始移动
- 🔇 **音效控制**：可随时开启或关闭游戏音效

## 🔧 自定义设置

### 速度档位说明（全新优化）
- **1档**：很慢 (900ms) - 新手入门
- **2档**：很慢 (800ms) - 熟悉操作
- **3档**：慢速 (700ms) - 基础练习
- **4档**：慢速 (600ms) - 进阶练习
- **5档**：中速 (500ms) - 默认标准速度 ⭐
- **6档**：中速 (400ms) - 熟练玩家
- **7档**：快速 (300ms) - 挑战模式
- **8档**：快速 (200ms) - 高手模式
- **9档**：很快 (100ms) - 极限挑战

### 音效设置
- **得分音效**：C-E-G大三和弦，营造愉快的得分体验
- **失败音效**：A4到A3的下降音调，提示游戏结束
- **音量控制**：固定音量0.3，适中不刺耳
- **静音模式**：完全关闭所有音效

## 🎮 控制说明

### 💻 电脑端操作
- **方向键控制**：↑↓←→ 控制蛇的移动方向
- **防反向**：不能直接反向移动（如向上时不能直接向下）
- **按键防抖**：防止过快按键导致的异常

### 📱 移动端操作
- **滑动控制**：在游戏区域内滑动手指
- **向上滑动**：蛇向上移动
- **向下滑动**：蛇向下移动
- **向左滑动**：蛇向左移动
- **向右滑动**：蛇向右移动
- **最小滑动距离**：30像素，避免误触
