# 🐍 贪吃蛇游戏

一个现代化的网页版贪吃蛇游戏，支持电脑端和移动端操作，具有友好的用户界面、智能的游戏机制、沉浸式音效体验和可爱的圆形蛇身设计。

## 🎮 游戏特色

### 🎯 核心功能
- **经典贪吃蛇玩法**：控制蛇移动，吃食物增长身体和得分
- **可爱圆形蛇身**：蛇身采用圆形设计，更加生动自然
- **智能蛇头眼睛**：蛇头有两个小眼睛，会根据移动方向转动
- **精致尾巴装饰**：蛇的最后一段身体后面有小尾巴，纯视觉效果
- **智能碰撞检测**：蛇头可以到达边界，只有穿过墙壁时才游戏结束
- **防误操作设计**：禁止反向移动，避免意外后退导致的游戏失败
- **网格显示**：清晰的网格线帮助玩家精确控制移动方向

### 🎚️ 游戏控制
- **10档速度调节**：从500ms到50ms，每档递减50ms，默认250ms
- **智能速度锁定**：游戏进行中锁定速度设置，防止意外调整
- **暂停/继续功能**：随时暂停游戏，支持中途休息
- **重新开始机制**：游戏结束后点击重新开始立即开始新游戏
- **即时开始**：点击开始游戏后蛇立即向右移动，无需等待输入
- **清除记录功能**：可以通过"清除最高纪录"按钮重置最高分

### 🔊 音效系统
- **得分音效**：吃到食物时播放愉快的和弦音效 (C-E-G)
- **游戏结束音效**：失败时播放下降音调的提示音
- **破纪录庆祝音效**：首次超过最高分时播放优美的胜利旋律
- **静音控制**：绿色静音按钮，随时开启/关闭音效
- **Web Audio API**：使用现代音频技术，无需外部音频文件

### 📱 多平台支持
- **电脑端**：使用方向键 ↑↓←→ 控制蛇的移动，空格键暂停/继续
- **移动端**：在游戏区域内滑动屏幕控制方向
- **响应式设计**：自动适配不同屏幕尺寸

## 🚀 快速开始

1. 打开 `index.html` 文件
2. 调整游戏速度（1档最慢500ms，6档默认250ms，10档最快50ms）
3. 点击"开始游戏"按钮，蛇立即开始向右移动
4. 使用方向键或滑动屏幕控制蛇的移动方向
5. 吃到红色食物增长身体和得分（+10分）
6. 点击绿色🔊静音按钮可开启/关闭音效
7. 点击红色🗑️清除最高纪录按钮可重置最高分

## 🎯 游戏规则

### 基本规则
- 🍎 **吃食物**：碰到红色食物可以增长身体和得分（+10分），同时播放得分音效和飘起分数动画
- 🚫 **避免自撞**：不能撞到自己的身体
- 🧱 **边界规则**：蛇头可以到达边界格子，但不能穿过墙壁
- ⚡ **移动限制**：不能反向移动，防止意外后退
- 🏆 **破纪录奖励**：首次超过最高分记录时播放优美的胜利旋律

### 游戏设置
- 🎚️ **速度调节**：游戏开始前可调整，游戏中被锁定
- ⏸️ **暂停功能**：游戏进行中可随时暂停/继续
- 🔄 **重新开始**：点击重新开始立即开始新游戏，按钮变为暂停
- ⚡ **即时开始**：点击开始游戏后蛇立即向右移动
- 🔇 **音效控制**：可随时开启或关闭游戏音效
- 🗑️ **清除记录**：可通过红色按钮重置最高分记录
